# Jogo De Adivinhacao

![](https://imgur.com/VOdpTTX.gif)

## Introdução

Este é um jogo cujo objetivo é adivinhar um número secreto de 1 a 20. O jogo possui um sistema de pontuação e 3 níveis de dificuldade.

## Funcionalidades

- **Dificuldades**: O jogo possui 3 níveis de dificuldade, sendo eles o fácil (10 tentativas), o normal (5 tentativas) e o difícil (3 tentativas).
- **Pontuação**: O jogo possui um sistema de pontuação, onde o jogador inicia a partida com 100 pontos e perde esses pontos a cada tentativa incorreta com base na diferença entre o número chutado e o número secreto.
- **Nova Partida**: A cada finalização de uma partida, o jogador pode escolher se deseja iniciar uma nova partida ou encerrar o jogo.
- **Números Chutados**: O jogo não permite a repetição de números chutados que estão incorretos.
- **Visualização**: O jogador pode visualizar sua pontuação e os números já chutados a cada tentativa.
